## 声明

Go 有 4 个主要的声明:

1. 变量（var）
2. 常量（cosnt）
3. 类型（type）
4. 函数（func）

### 短变量声明

```go
func main(){
  a := 1
}
```

短变量声明是个局部变量

### 指针

- 变量是存储值得地方。
- 指针的值是一个变量的**地址**
- 一个指针，指示，值所保存的位置
- 不是所有的值都有地址，但是所有的变量都有
- 使用指针，可以在无需知道变量名字的情况下，间接读取或更新变量的值。

```go
x := 1
p := &x // &x 是 x 的地址
fmt.Println(*p) // 1
*p = 2
fmt.Println(x)  // 2
```

### new 函数

```go
p := new(int)
fmt.Println(*p) // 0
*p = 2
fmt.Println(*p) // 2
```

`new(T)` 初始化为 T 类型的零值，并且返回其**地址**.
